# Hospital Management System (HMS)
**Project ID: DBS25F238**

A comprehensive Hospital Management System developed as a term project for CMPE-341 Database Systems course.

##  Features

### Patient Management
- Add, edit, and delete patient records
- Track patient medical history
- Blood type and contact information management

### Doctor Management
- Doctor directory with specialization
- Department-wise organization
- Contact and scheduling information

### Appointment Scheduling
- Book appointments with doctors
- Track appointment status
- Date and time management

### Reports Generation
- Patient list reports
- Doctor directory reports
- Appointment summaries
- PDF report generation using iText library

### Database Features
- 15+ normalized database tables
- 5 database views for complex queries
- 3 stored procedures for business logic
- 2 triggers for automated operations
- 10+ constraints for data integrity
- Transaction management

## 🛠️ Technology Stack

**Backend:**
- Java (JDBC, Servlets)
- MySQL Database
- iText PDF Library

**Frontend:**
- HTML5 & CSS3
- JavaScript (Vanilla)
- Responsive Design

**Tools:**
- VS Code
- Git & GitHub
- phpMyAdmin
